# Happy Habits
An iOS app custom designed to track habits and committing to lifelong habits!

# Why Did I build this?
It started when I wanted to develope healthy habits and while there are many habit apps across the Apple App Store, many required paid subscriptions or features I don't neccessarily want to use. As a result, I got a brainwave. Why not learn the SwiftUI and build an app with features that I know will help me and I have control of the features I coded. It also had been a longtime dream of learning Swift as long as 2017 when I learned to code small Swift apps using the Swift Playground app on iPad.

# Current Feature roadmap
This version of the app now allows the ability to add and remove habits and self resets habits with each new rolling day. There is currently no plans to add additional features at this moment, just UI updates and bug fixes.
